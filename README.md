# 直播答题 PWA

基于抖音弹幕监听的直播答题 PWA 应用，9:16 竖屏布局，支持题目管理、弹幕互动、实时排行，可离线使用。

## 功能特性

- **9:16 竖屏直播布局**：头部（直播间名称 + 直播画面预留）→ 题目 → ABCD 选项 → 倒计时 → 排行榜
- **题目管理**：CSV/JSON 批量导入导出、批量删除、按科目/关键词条件查询
- **直播间配置**：直播间名称、ID、Cookies、WebSocket 地址端口
- **弹幕答题**：观众在直播间发送 A/B/C/D 即可参与答题
- **实时排行榜**：答对得分（基础 10 分 + 倒计时剩余奖励），自动排序
- **本地持久化**：所有配置与题库存储于 localStorage
- **PWA 离线支持**：Service Worker 缓存核心资源，可安装到桌面

## 如何设置

### 1. 配置直播间

打开应用后点击右上角 **⚙ 设置** 按钮，进入设置页面：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 直播间名称 | 用于主页头部展示 | `知识答题直播间` |
| 直播间 ID | 抖音直播间 ID | `516466932480` |
| Cookies | 选填，用于鉴权 | （留空也可） |
| WebSocket 地址 | 本地弹幕服务地址 | `127.0.0.1` |
| 端口 | 本地弹幕服务端口 | `1088` |

填写完成后点击 **保存配置**，可点击 **测试连接** 验证弹幕服务是否可达。

### 2. 导入题目

在设置页的「题目管理」区域：

- 点击 **导入 CSV** 或 **导入 JSON** 上传题库
- 也可拖拽 `示例题库.csv` 进行导入测试

题目字段如下（CSV 表头需与此一致）：

| 字段 | 说明 |
|------|------|
| 科目 | 题目分类，如「常识」「地理」 |
| 序号 | 题目编号 |
| 题目 | 题干内容 |
| 选项A / 选项B / 选项C / 选项D | 四个选项 |
| 正确答案 | A / B / C / D |

> 导入时按「科目 + 序号」自动去重，重复题目会被跳过。

### 3. 启动弹幕服务（可选）

弹幕答题功能依赖本地 WebSocket 弹幕服务（默认 `ws://127.0.0.1:1088`）。若仅需查看题目与界面，可跳过此步；若需要观众实时答题，请先启动抖音弹幕转发服务。

## 如何启动

### 方式一：Python 内置服务器（推荐）

```bash
cd /Users/yingbin/Desktop/codex_projects/dati
python3 -m http.server 8080
```

浏览器访问：<http://localhost:8080>

### 方式二：Node 静态服务器

```bash
cd /Users/yingbin/Desktop/codex_projects/dati
npx serve -l 8080
# 或
npx http-server -p 8080
```

浏览器访问：<http://localhost:8080>

### 安装为 PWA（桌面/手机）

1. 用 Chrome / Edge 打开 <http://localhost:8080>
2. 点击地址栏右侧的 **安装** 图标（或菜单 → 安装应用）
3. 安装后即可从桌面独立启动，支持离线使用

## 使用流程

1. 进入 **设置** → 填写直播间信息 → 保存
2. 导入题库（CSV/JSON）
3. 返回主页 → 浏览题目（上一题 / 下一题）
4. 点击 **开始答题** → 倒计时启动（默认 15 秒）
5. 观众在直播间发送 `A` / `B` / `C` / `D` 作答
6. 倒计时结束 → 显示正确答案与各选项作答人数
7. 排行榜实时更新，可点击 **重置** 清空积分

## 计分规则

- 答对：基础 **10 分** + 倒计时剩余时间奖励（剩余越多得分越高）
- 答错：不扣分
- 每人每题仅首次作答有效

## 题库格式示例

**CSV 格式：**

```csv
科目,序号,题目,选项A,选项B,选项C,选项D,正确答案
常识,1,中国的首都是哪个城市？,上海,北京,广州,深圳,B
地理,3,长江发源于哪个高原？,内蒙古高原,青藏高原,云贵高原,黄土高原,B
```

**JSON 格式：**

```json
[
  {
    "科目": "常识",
    "序号": "1",
    "题目": "中国的首都是哪个城市？",
    "选项A": "上海",
    "选项B": "北京",
    "选项C": "广州",
    "选项D": "深圳",
    "正确答案": "B"
  }
]
```

> 也支持英文字段名：`subject`、`seq`、`question`、`a`/`b`/`c`/`d`、`answer`。

## 技术说明

- 纯前端实现，无需后端数据库，数据存储于浏览器 localStorage
- PWA 通过 Service Worker (`sw.js`) 缓存核心资源实现离线可用
- 弹幕接入依赖 `DouyinLiveWS.js` 连接本地 WebSocket 服务
- 兼容桌面与移动端浏览器，推荐 Chrome / Edge

## 常见问题

**Q：页面显示「未配置直播间」？**
A：进入设置页填写直播间名称和 ID 并保存。

**Q：状态显示「连接断开，重连中」？**
A：请确认本地 1088 端口的抖音弹幕转发服务已启动。

**Q：导入题目后主页仍显示「暂无题目」？**
A：刷新主页即可加载最新题库。

**Q：如何重置排行榜？**
A：点击排行榜标题栏右侧的 **重置** 按钮。
